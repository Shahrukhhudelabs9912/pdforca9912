## Excel PDF Ke Liye Best Practices

### Print Area Set Karo (Excel Mein)

Convert karne se pehle Excel mein print area set karo:
1. Jo cells PDF mein chahiye unhe select karo
2. `Page Layout` → `Print Area` → `Set Print Area`
3. Ab sirf selected area PDF mein aayega

### Page Orientation Choose Karo

Wide tables ke liye Landscape better hoti hai:
- Excel mein: `Page Layout` → `Orientation` → `Landscape`
- Phir PDFOrca se convert karo

### Fit to Page

Agar table ek page pe fit nahi ho rahi:
- Excel mein: `Page Layout` → `Scale to Fit` → Width: 1 page, Height: Automatic

### Headers aur Footers

Agar Excel mein headers/footers set hain (company name, page numbers), wo PDF mein bhi aayenge.

## Common Problems Aur Solutions

### Problem: Columns Cut Ho Rahi Hain

**Cause:** Table page width se zyada wide hai.

**Solutions:**
1. Excel mein column width kam karo
2. Landscape orientation use karo
3. Font size thoda kam karo (10pt se 9pt)
4. `Scale to Fit` use karo — `Page Layout` → `Scale to Fit`

### Problem: Multiple Sheets Mein Se Sirf Ek Sheet Convert Hui

**Cause:** PDFOrca by default active sheet convert karta hai.

**Solution:** Excel mein sab sheets select karo (Ctrl+Click on sheet tabs), phir convert karo. Ya har sheet alag convert karo aur [PDFOrca Merge PDF](https://pdforca.com/merge-pdf) se combine karo.

### Problem: Charts Blurry Aa Rahe Hain

**Cause:** Low resolution conversion.

**Solution:** PDFOrca high quality conversion use karta hai — agar phir bhi issue ho, Excel mein chart size badha do pehle.

### Problem: Formulas Dikh Rahe Hain Values Ki Jagah

**Cause:** Excel file mein formulas show ho rahi hain.

**Solution:** Excel mein `Ctrl+`` (backtick) dabao — ye formula view toggle karta hai. Values view mein convert karo.

### Problem: File Size Bahut Badi Hai

**Cause:** Charts aur images wali Excel files badi PDF banati hain.

**Solution:** Convert ke baad [PDFOrca Compress PDF](https://pdforca.com/compress-pdf) se compress karo.
