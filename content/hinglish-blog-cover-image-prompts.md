# Hinglish Blog Cover Image Prompts (DALL-E / ChatGPT Plus)

**Size:** 1200x630px
**Format:** WebP (har prompt mein specified hai — DALL-E se download ke baad squoosh.app se convert karo)
**Style:** Consistent blue gradient, 3D icons, minimal text (English blogs jaisa hi — brand consistency ke liye)
**Save location:** `frontend/public/blog/covers/[slug].webp`
**Frontmatter field:** `cover: "/blog/covers/[slug].webp"`

> ⚠️ **Note:** Code (`frontend/lib/blog.ts`) frontmatter mein `cover:` field padhta hai, `coverImage:` nahi. Isliye har blog `.md` mein `cover: "/blog/covers/[slug].webp"` add karo.

**Status Legend:**
- [ ] = Image NOT created yet
- [x] = Image DONE

**Ye file kis liye hai:** English blogs ke liye `content/blog-cover-image-prompts.md` hai. Ye uska Hinglish version hai — 60-day Hinglish content calendar (`content/hinglish-blog-calendar-60days.md`) ke saare 26 posts ke DALL-E cover prompts yahan hain.

## Tips

1. **WebP format:** Har prompt mein "Output as WebP" likha hai — DALL-E WebP generate kar dega
2. **Agar DALL-E PNG deta hai:** squoosh.app pe free mein WebP mein convert karo
3. **Consistency:** Sab images same blue gradient background — English blogs jaisa hi, brand identity ke liye
4. **DALL-E text likhne mein weak hai:** Hinglish/Hindi text image ke andar mat maango — baad mein Canva/Figma se add karo agar chahiye
5. **File save karo:** `frontend/public/blog/covers/[slug].webp`
6. **Frontmatter update karo:** Blog `.md` file mein `cover: "/blog/covers/[slug].webp"` add karo (field ka naam `cover` hai, `coverImage` nahi)
7. **Jab image ban jaye:** Is file mein `[ ]` ko `[x]` mein change karo aur Progress Tracker update karo

---

## Cover Kaise Add Kare — Step by Step

1. Upar diya gaya prompt copy karo (jis blog ka cover chahiye)
2. ChatGPT Plus (DALL-E) mein paste karke image generate karo
3. Image download karo. Agar PNG mile toh squoosh.app se WebP mein convert karo
4. File ko rename karo `[slug].webp` (exactly slug jaisa, upar table mein diya hai)
5. Save karo yahan: `frontend/public/blog/covers/[slug].webp`
6. Blog ki `.md` file (`content/blog/[slug].md`) ke frontmatter mein ye line add karo:
   ```yaml
   cover: "/blog/covers/[slug].webp"
   ```
7. Is file mein us post ka status `[ ]` → `[x]` kar do
