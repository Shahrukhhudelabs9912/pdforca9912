## Page Numbering Ke Tips

**Cover page skip karna:** Agar pehla page cover/title page hai, starting number 1 rakho lekin usually cover page pe number nahi hota. Kuch tools first page skip option dete hain.

**Consistent position:** Ek document mein hamesha same position use karo — mix mat karo.

**Font size:** Page numbers subtle hone chahiye — 10-12pt ideal hai. Bahut bada ugly lagta hai.

**Merged documents:** Pehle merge karo, phir number lagao — taaki continuous numbering ho.

**Print consideration:** Agar print karna hai toh margins mein number rakho — content overlap na ho.

## Frequently Asked Questions

**Q1: Kya page numbers document ke content ko cover karenge?**
Nahi, page numbers margins mein add hote hain — content ke bahar. Bottom center/right position content ko overlap nahi karti. Preview mein check kar sakte ho.

**Q2: Kya pehle page (cover) pe number nahi lagana possible hai?**
Kuch cases mein first page skip kar sakte ho. Ya starting number set karke manage kar sakte ho. Agar cover page number-free chahiye, use pehle [PDFOrca Split PDF](https://pdforca.com/split-pdf) se alag karo, baaki pages number karo, phir merge karo.

**Q3: Page numbers ki position baad mein change kar sakte hain?**
Directly nahi. Original file se dobara numbering apply karo alag position ke saath. Isliye original file save rakho.

**Q4: "Page X of Y" format mein Y automatically calculate hota hai?**
Haan, total pages (Y) automatically calculate hota hai. Agar 30-page PDF hai toh har page pe "Page 1 of 30", "Page 2 of 30" aayega.

**Q5: Kya numbers ki font aur color change kar sakte hain?**
Basic customization available hai — size aur position. Color usually black hota hai (standard). Advanced formatting ke liye document editor use karo.
