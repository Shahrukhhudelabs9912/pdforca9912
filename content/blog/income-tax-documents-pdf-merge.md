ITR filing ka time aata hai toh sabse badi problem hoti hai — documents scattered hain. Form 16 alag jagah hai, bank statements alag, investment proofs alag, aur CA ko sab ek saath bhejna hai. Baar baar alag files attach karna, email mein limit aana — ye sab frustrating hai.

Solution simple hai: **sab documents ek PDF mein merge karo** aur ek hi file bhejo. CA ko bhi easy hoga, aur tumhara kaam bhi organized rahega.

Is guide mein main bataunga ki ITR filing ke liye kaunse documents chahiye, unhe kaise organize karo, aur PDFOrca se ek clean merged PDF kaise banao.

## ITR Filing Ke Liye Kaunse Documents Chahiye

Salary wale employees ke liye:
- **Form 16** — employer se milta hai (Part A aur Part B)
- **Bank statements** — sab accounts ke (April to March)
- **Form 26AS / AIS** — Income Tax portal se download karo
- **Investment proofs** — PPF, ELSS, LIC, NPS receipts
- **Home loan certificate** — agar hai toh
- **Rent receipts** — HRA claim ke liye
- **Medical insurance premium receipt** — 80D ke liye

Freelancers/self-employed ke liye upar wale ke saath:
- **Invoices** ya income proof
- **Business expense receipts**
- **TDS certificates** (Form 16A)

## Documents Merge Karne Se Pehle — Preparation

### Step 1: Sab Documents Collect Karo

Ek folder banao — physical ya digital — aur sab documents ek jagah rakho. Missing documents pehle download karo:
- Form 26AS: [incometax.gov.in](https://www.incometax.gov.in) → Login → e-File → Income Tax Returns → View Form 26AS
- AIS: Same portal → Services → Annual Information Statement

- [PDFOrca Word to PDF](https://pdforca.com/word-to-pdf)
- [PDFOrca Excel to PDF](https://pdforca.com/excel-to-pdf)

### Step 3: File Size Check Karo

Email attachment limit usually 10-25MB hoti hai. Agar individual files badi hain, pehle compress karo:
- [PDFOrca Compress PDF](https://pdforca.com/compress-pdf)

## CA Ko Bhejne Ke Liye Tips

**Email se bhejna hai:**
- File size 10MB se kam rakho — compress karo agar zyada hai
- Subject line clear rakho: "ITR Documents FY 2025-26 — [Tumhara Naam]"
- Email body mein briefly list karo ki kya-kya include hai

**WhatsApp se bhejna hai:**
- WhatsApp pe 100MB tak file bhej sakte ho
- Lekin sensitive documents WhatsApp pe bhejne se pehle soch lo — end-to-end encrypted hai, lekin phir bhi

**Google Drive / Dropbox:**
- Badi files ke liye best option
- Link share karo CA ke saath
- Folder access sirf CA ko do


## Common Mistakes Jo Avoid Karni Chahiye

**Galat financial year ke documents:** FY 2025-26 ke liye April 2025 se March 2026 ke documents chahiye. Purane documents mat include karo.

**Form 16 incomplete:** Form 16 mein Part A aur Part B dono hone chahiye. Sirf ek part se kaam nahi chalega.

**Bank statement incomplete:** Poore 12 months chahiye — April se March. Kuch log sirf 3-6 months ki statement dete hain jo insufficient hoti hai.

**File size bahut badi:** 50MB+ ki file email nahi hogi. Compress karo ya Drive link use karo.

**Password protected files merge nahi hongi:** Pehle [PDFOrca Unlock PDF](https://pdforca.com/unlock-pdf) se password hatao.

## Frequently Asked Questions

**Q1: Kitne documents ek PDF mein merge kar sakte hain?**
PDFOrca mein koi limit nahi hai documents ki count pe. File size 100MB se kam honi chahiye total. ITR documents usually 5-20MB ke beech hote hain — koi problem nahi hogi.

**Q2: Kya merge karne ke baad documents edit ho sakte hain?**
Merged PDF mein directly edit karna mushkil hai. Agar koi document galat add ho gaya, original files se dobara merge karo. Isliye original files alag save rakho.

**Q3: CA ne specific format maanga hai — kya PDFOrca se wo format milega?**
PDFOrca standard PDF banata hai jo har jagah open hoti hai. Agar CA ne koi specific software ya format maanga hai, unse clarify karo. Usually standard PDF hi acceptable hoti hai.

**Q4: Kya password laga sakte hain merged PDF pe?**
Haan. Merge karne ke baad [PDFOrca Protect PDF](https://pdforca.com/protect-pdf) se password laga sakte ho. Sensitive financial documents pe password lagana good practice hai.

**Q5: Mobile se bhi merge kar sakte hain?**
Haan, mobile browser mein PDFOrca perfectly kaam karta hai. Files app ya Google Drive se PDF select karo, merge karo, download karo.
