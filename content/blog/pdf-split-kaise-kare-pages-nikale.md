## PDF Split Karne Ke Tips

**Pehle page numbers check karo:** PDF reader mein file kholo aur note karo ki tumhe exactly kaunse pages chahiye. Galat pages extract karne se time waste hoga.

**Range format sahi use karo:** 
- Single page: `5`
- Multiple pages: `5, 10, 15`
- Range: `5-15`
- Mix: `1-5, 10, 20-25`

**Large files ke liye:** Agar file 50MB se zyada hai, thoda time lagega upload mein. Stable internet connection pe karo.

**ZIP file unzip karna:** Windows mein ZIP file pe right-click karo → "Extract All". Mobile pe koi file manager app use karo.

## PDF Split vs PDF Extract — Kya Fark Hai

Ye dono terms aksar confuse karte hain:

**Split:** Ek PDF ko multiple PDFs mein todna — jaise 100-page file ko 4 parts of 25 pages each.

**Extract:** Specific pages nikalna — jaise 100-page file mein se sirf pages 10, 20, 30 nikalna.

## Split Ke Baad File Size

Jab tum pages extract karte ho, file size proportionally kam ho jaati hai. 100-page 10MB file mein se 10 pages nikale toh roughly 1MB ki file banegi. Exact size depend karta hai content pe — images wale pages zyada space lete hain.

Agar extracted file bhi badi lag rahi ho, [PDFOrca Compress PDF](https://pdforca.com/compress-pdf) se aur compress kar sakte ho.

## Common Mistakes Aur Unke Solutions

**Galat pages extract ho gayi:** Undo nahi hota, lekin original file safe hai — dobara try karo sahi page numbers ke saath.

**ZIP file nahi khul rahi:** Windows 10/11 mein built-in unzip hai. Older Windows mein WinRAR ya 7-Zip use karo.

**Pages out of order aa rahi hain:** Page range mein order matter karta hai — `5-10` aur `10-5` alag results denge.

**File upload nahi ho rahi:** File size check karo — 100MB se zyada nahi honi chahiye. Badi file pehle compress karo.

Nahi. PDFOrca pe upload ki gayi file process hoti hai aur result download hota hai — tumhari original file tumhare device pe safe rehti hai. PDFOrca servers pe bhi files automatically delete ho jaati hain processing ke baad.
