Mobile se document scan kiya aur PDF bana li — lekin khola toh page sideways tha ya ulta. Ya phir kisi ne email mein PDF bheji jisme kuch pages landscape mein hain aur kuch portrait mein — padhne ke liye baar baar phone ghuma rahe ho. Ye problem bahut common hai aur solution bilkul simple hai.

**PDF rotate karna** matlab hai — PDF ke pages ki orientation change karna. 90 degree, 180 degree, ya 270 degree — jitna chahiye utna ghuma sakte ho. PDFOrca pe ye kaam free mein, bina kisi software ke, seedha browser mein hota hai.

## Kab Padti Hai PDF Rotate Karne Ki Zaroorat

**Scanned documents:**
- Mobile camera se scan ki document sideways aa gayi
- Scanner ne landscape mein scan kar diya jab portrait chahiye tha
- Old documents scan karte waqt orientation galat ho gayi

**Received PDFs:**
- Kisi ne galat orientation mein PDF bheji
- Multiple pages hain — kuch seedhe hain, kuch ulte
- Government document download kiya jo sideways hai

**Presentations aur reports:**
- Kuch pages landscape mein hain (charts/tables ke liye) aur kuch portrait mein
- Print karne se pehle sab pages same orientation mein chahiye

### Step 4: Rotation Angle Choose Karo

- **90° Clockwise** — page ko right side ghuma do
- **90° Counter-clockwise** — page ko left side ghuma do
- **180°** — page bilkul ulta kar do (upside down fix karne ke liye)

Agar page sideways (landscape) hai aur seedha (portrait) karna hai — 90° clockwise ya counter-clockwise try karo. Preview dekho kaunsa sahi lag raha hai.

### Step 5: Apply Karo Aur Download Karo

"Rotate PDF" button dabao. Processing hogi aur rotated PDF download ke liye ready ho jaayegi.

## Rotation Angles Ka Matlab — Simple Guide

Socho tumhara page ek ghadi ki tarah hai:

| Problem | Solution |
|---------|----------|
| Page right side jhuka hua hai | 90° Counter-clockwise |
| Page left side jhuka hua hai | 90° Clockwise |
| Page bilkul ulta hai | 180° |
| Page seedha hai par print ulta aata hai | 180° |

Agar pehli baar galat ho jaaye — koi baat nahi. Original file safe hai, dobara try karo.

## Frequently Asked Questions

**Q1: Rotate karne se PDF ki quality kharab hoti hai?**
Bilkul nahi. Rotation sirf orientation change karta hai — text, images, fonts sab exactly same quality mein rehte hain. Koi compression ya quality loss nahi hota.

**Q2: Kya ek baar mein sab pages alag-alag angles pe rotate kar sakte hain?**
Haan. Pehle ek set of pages select karo, ek angle apply karo. Phir doosra set select karo, doosra angle apply karo. Ya ek baar mein sab same angle pe rotate karo.

**Q3: Rotate ki gayi PDF wapas original orientation mein aa sakti hai?**
Haan — original file tumhare device pe safe hai. Rotated PDF download ki hai, original unchanged hai. Agar galat rotate ho gayi toh original se dobara karo.

**Q4: Mobile se scan ki PDF automatically rotate ho sakti hai?**
Kuch scanner apps (Adobe Scan, Microsoft Lens) automatically orientation detect karte hain. Lekin agar phir bhi galat ho toh PDFOrca se manually fix karo.

**Q5: Kya sirf ek page wali PDF bhi rotate ho sakti hai?**
Haan, single page PDF bhi rotate hoti hai — same process follow karo.
