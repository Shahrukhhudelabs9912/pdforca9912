## Conversion Ke Baad Data Clean Karna

Aksar converted Excel mein thodi cleaning karni padti hai:

**Extra spaces:** `TRIM()` function use karo — `=TRIM(A1)`

**Numbers text ke roop mein:** Column select karo → Data → Text to Columns → Finish. Ya `VALUE()` function.

**Date format:** `DATEVALUE()` function ya Format Cells se date format set karo.

**Merged cells:** Unmerge karo aur data fill karo manually.
