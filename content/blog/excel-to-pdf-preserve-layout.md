You spent two hours perfecting a spreadsheet — aligned columns, conditional formatting, a chart that finally tells the story. Then you email it and the recipient opens it in a different Excel version. Columns shift. Colours vanish. The chart floats to the wrong page.

## Why Convert Excel to PDF?

Spreadsheets are built for editing. PDFs are built for sharing. When your spreadsheet needs to reach someone else, PDF is almost always the right format.

- **Consistent appearance** — The recipient sees exactly what you see. No version mismatches, no missing fonts, no shifted columns.
- **Universal compatibility** — PDFs open on Windows, Mac, Linux, iOS, Android, and Chromebooks without Excel installed. Every browser can display a PDF.
- **Professional presentation** — Financial reports, invoices, timesheets, and budgets look polished as PDFs. A raw `.xlsx` file feels like a draft.
- **Protection** — Recipients can view and print, but they cannot edit your formulas, change your numbers, or accidentally break your formatting.
- **Smaller file size** — Complex spreadsheets with embedded objects can be surprisingly large. PDFs are typically 50-80% smaller.
- **Compliance** — Many organizations, auditors, and government agencies require submissions in PDF format, not editable spreadsheets.

## What Gets Preserved — and What Doesn't

Understanding what transfers to PDF helps you prepare your spreadsheet before converting.

| Element | Preserved in PDF? | Notes |
|---------|-------------------|-------|
| Column widths and row heights | Yes | Exact dimensions maintained |
| Cell formatting (bold, italic, colours) | Yes | All font styling transfers |
| Borders and gridlines | Yes | Both custom borders and gridlines |
| Merged cells | Yes | Layout stays intact |
| Conditional formatting | Yes | Colours and icons as they appear on screen |
| Charts and graphs | Yes | Rendered as they display in Excel |
| Images and shapes | Yes | Position and size maintained |
| Formulas | Values only | The calculated result appears, not the formula itself |
| Hyperlinks | Depends | Some converters preserve clickable links |
| Comments and notes | No | Strip these before converting, or screenshot them separately |
| Macros and VBA | No | Code does not transfer (and shouldn't — PDFs are static) |
| Data validation dropdowns | No | Only the selected value appears |

**Key insight:** PDF captures the *visual state* of your spreadsheet at the moment of conversion. Anything you can see on screen will appear in the PDF. Anything hidden (formulas behind cells, macros, data validation rules) will not.

## Preparing Your Spreadsheet for the Best PDF Output

A few minutes of preparation can dramatically improve your PDF output.

### 1. Set the Print Area

Excel converts the **print area** to PDF. If you haven't set one, the tool converts all data on the sheet — which may include empty rows, scratch calculations, or data dumps you don't want in the final document.

To set the print area in Excel: select the range you want → Page Layout → Print Area → Set Print Area.

### 2. Adjust Page Breaks

Open Print Preview (Ctrl+P) and check where pages break. Long spreadsheets often split mid-table or cut a chart in half. Drag page breaks to logical positions — between sections, after totals, before a new data group.

### 3. Choose Orientation

- **Portrait** for narrow spreadsheets (2-5 columns) — looks like a standard document
- **Landscape** for wide spreadsheets (6+ columns) — prevents columns from being cut off or squeezed

### 4. Scale to Fit

If your spreadsheet is slightly too wide for one page, use the "Fit All Columns on One Page" scaling option instead of manually shrinking columns. This maintains readability while ensuring nothing gets cropped.

### 5. Add Headers and Footers

Professional PDFs include context: the report title in the header, page numbers in the footer, and the date of generation. Set these in Excel before converting — they carry over to the PDF.

### 6. Hide Unnecessary Sheets

If your workbook has multiple sheets but you only need one in the PDF, hide the others or select only the target sheet before converting.

## Handling Multi-Sheet Workbooks

When your Excel file contains multiple sheets, the converter processes all visible sheets sequentially. Each sheet becomes a new section in the PDF.

**Tips for multi-sheet conversion:**

- **Order matters** — sheets appear in the PDF in the same tab order as your workbook. Rearrange tabs before converting.
- **Consistent formatting** — use the same fonts, header styles, and page orientation across sheets for a cohesive document.
- **Page breaks between sheets** — each sheet starts on a new page, so your summary sheet won't run into the detail sheet.

If you only need one sheet, either hide the others or copy the target sheet to a new workbook and convert that instead.

## Common Formatting Issues and Fixes

| Problem | Cause | Fix |
|---------|-------|-----|
| Columns cut off on the right | Spreadsheet wider than page | Switch to landscape, or use "Fit All Columns on One Page" |
| Tiny, unreadable text | Too many columns forced onto one page | Reduce the number of columns per page, or use a larger paper size (A3) |
| Chart appears on a separate page | Chart object extends beyond the print area | Resize the chart to fit within the data region |
| Blank pages in the PDF | Empty cells or print area extends beyond data | Reset the print area to cover only the data range |
| Gridlines missing | Gridlines not enabled for printing | In Excel: Page Layout → check "Print" under Gridlines |
| Header/footer not showing | Not set in Page Setup | Add headers/footers in Page Layout → Page Setup → Header/Footer |

## FAQ

**Does converting remove my formulas?**
The PDF shows calculated values, not formulas. If cell B5 contains `=SUM(B2:B4)` and displays 1,500 — the PDF shows "1,500." The formula itself is not visible or accessible. This is actually a feature for most use cases — you share the results without exposing your calculation logic.

**Can I convert a password-protected Excel file?**
You'll need to remove the password first. Open the file in Excel, go to File → Info → Protect Workbook → remove the password, save, then upload the unprotected version.

**What if my spreadsheet has multiple sheets?**
All visible sheets are converted. Each sheet starts on a new page in the PDF. Hide any sheets you don't want included.

**Will my charts look the same?**
Yes. Charts are rendered as they appear on screen — same colours, same axis labels, same data points. They become static images in the PDF, which means they display identically everywhere.

**Can I convert `.csv` files too?**
CSV files don't contain formatting — they're plain text with commas. Open the CSV in Excel first, apply your formatting (headers, column widths, borders), then save as `.xlsx` and convert to PDF. Converting a raw CSV produces a plain, unformatted table.

**What's the maximum file size?**
The tool handles files up to 100 MB. For very large workbooks, consider splitting into separate files by sheet or section before converting.

**Can I convert back from PDF to Excel?**
Yes — use the [PDF to Excel](/pdf-to-excel) tool. Note that converting back extracts data from tables, but some formatting may differ from the original. For best results, keep the original `.xlsx` file as your editable master copy and the PDF as the distribution version.
