## Common Problems Aur Solutions

### Problem: Fonts Change Ho Gayi PDF Mein

**Cause:** Word mein non-standard font use ki thi.

**Solution:** PDFOrca fonts embed karta hai. Lekin agar phir bhi issue ho:
- Word mein `File → Options → Save → Embed fonts in the file` check karo
- Ya standard fonts use karo: Calibri, Arial, Times New Roman

### Problem: Images Shift Ho Gayi

**Cause:** Images "In Line with Text" ke bajaye floating hain.

**Solution:** Word mein images select karo → `Wrap Text → In Line with Text` set karo, phir convert karo.

### Problem: Table Page Se Bahar Ja Rahi Hai

**Cause:** Table width page width se zyada hai.

**Solution:** Word mein table select karo → `Table Properties → Preferred width → 100%` set karo.

### Problem: Extra Blank Page Aa Gayi

**Cause:** Document ke end mein extra empty paragraphs hain.

**Solution:** Word mein `Ctrl+End` dabao, extra blank lines delete karo, phir convert karo.

### Problem: PDF Bahut Badi Hai

**Cause:** High-resolution images hain document mein.

**Solution:** Convert ke baad [PDFOrca Compress PDF](https://pdforca.com/compress-pdf) se compress karo.

## Word Mein Formatting Tips Jo PDF Ko Better Banate Hain

**Styles use karo:** Heading 1, Heading 2, Normal — ye PDF mein proper structure dete hain aur bookmarks bhi ban jaate hain.

**Page breaks explicit rakho:** `Ctrl+Enter` se page break do — automatic page breaks kabhi kabhi PDF mein shift ho jaate hain.

**Images inline rakho:** Floating images PDF mein shift ho sakti hain — "In Line with Text" safe hai.

**Margins check karo:** `Page Layout → Margins` — standard margins (1 inch) use karo.

**Headers/footers:** Agar page numbers ya company name header mein hai, PDF mein bhi aayega.

## Frequently Asked Questions

**Q1: Kya Word ka koi bhi version convert ho sakta hai?**
Haan, .doc (older) aur .docx (newer) dono convert hote hain. .docx better results deta hai — agar .doc hai toh pehle Word mein .docx mein save karo.

**Q2: Convert ke baad PDF edit kar sakte hain?**
PDF directly edit karna mushkil hai. Agar changes karne hain, original Word file edit karo aur dobara convert karo. Ya [PDFOrca PDF to Word](https://pdforca.com/pdf-to-word) se wapas Word mein convert karo.

**Q3: Kya tracked changes PDF mein dikh jaate hain?**
Agar Word mein tracked changes accept/reject nahi kiye, wo PDF mein dikh sakte hain. Convert se pehle `Review → Accept All Changes` karo.

**Q4: Password protected Word file convert hogi?**
Nahi directly. Pehle Word mein file kholo (password daalo), save karo bina password ke, phir PDFOrca se convert karo.

**Q5: Multiple Word files ek PDF mein kaise banaye?**
Har file alag convert karo, phir [PDFOrca Merge PDF](https://pdforca.com/merge-pdf) se combine karo.
